# ALU Launch 🚀

ALU Launch is a Flutter mobile application that connects ALU students with student-led startups by helping students discover and apply for internship opportunities.

## Links

- 📄 Technical Report: https://docs.google.com/document/d/1xOVcbpKMWA1CBga3gwhfMKH8QFibb55ZJ6NG32bK4u0/edit?usp=sharing
- 🎥 Demo Video: https://drive.google.com/file/d/1RIhkmrwieyuFE5BNgvNhm0V5Jg3n8ZpY/view?usp=sharing

## Tech Stack

- Flutter & Dart
- Firebase Authentication
- Cloud Firestore
- Riverpod

## Run the App

1. Clone the repository:

```bash
git clone https://github.com/EmmanuelDufitumukiza/alu_launch
cd alu_launch
````

2. Install dependencies:

```bash
flutter pub get
```

3. Configure Firebase:

```bash
flutterfire configure
```

4. Run the app:

```bash
flutter run
```
